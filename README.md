## FAQ

### HKG.T1.Starter 的流量超出后怎么处理？

超出月流量上限后，机器会被限速而不是直接停机。限速后仍然可以使用，只是带宽会降到一个较低的水平。如果业务流量不稳定，这个机制比直接断网友好一些。

### CN2 GIA 和普通 CN2 GT 差在哪？

GIA（Global Internet Access）是电信的高端商用线路，全程走专用通道，不和普通用户流量混跑。GT（Global Transit）则是共享线路，晚高峰容易拥堵。HKG.T1 走的是 GIA，这是它比同价位普通香港 VPS 贵的主要原因。

### 香港节点对移动用户友好吗？

HKG.T1 系列对移动用户走 CMI 线路，比走绕路方案的机器要好。不过移动用户如果主要在华南地区，延迟表现会比电信用户更明显地受益；华北用户延迟会稍高一些，这是地理距离决定的，不是线路问题。

### 可以跑 IPv6 吗？

DMIT HKG.T1 套餐提供 IPv4 地址，IPv6 支持情况建议在下单前通过工单确认，官网套餐页面会有最新说明。

### 机器支持自定义 ISO 安装吗？

标准套餐提供常见 Linux 发行版（Debian、Ubuntu、CentOS 等）的一键安装，自定义 ISO 需要联系客服确认是否支持，不同套餐政策可能不同。

### 套餐可以随时升级吗？

可以。在控制面板内提交升级申请，按差价补款后生效，数据一般可以保留。具体流程以工单回复为准。
